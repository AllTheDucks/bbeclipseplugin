New Building Block Wizard

This wizard enables you to create a new Building Block from a number of template and sample projects. The Purpose of the fields is as follows:

Vendor ID This is an internal ID. Together with the building block "handle" it uniquely identifies your plugin within blackboard. Usually it is the initials or abbreviation of your institution's name.
e.g. mit, qut, ucla, etc. Maximum Length 4 Characters.

Plugin Handle Stored in bb-manifest.xml in the handle element. The internal name of your building block, e.g. video-player, sms-tool, feedback-form, etc, etc. Maximum Length 32 Characters.

Base Package The base java package for java source files in this project. e.g. edu.ucla.mycustomtool

Development Server The development server running blackboard. You should have the "Starting Block" Building block running on this server for remote deployment to work.

Target Blackboard Version The blackboard API version that your code will be compiled against.

Project Template The template which your project will be based on.
