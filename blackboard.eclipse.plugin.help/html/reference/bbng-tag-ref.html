bbNG Tag Library Reference

From Blackboard v 9.0, the bbNG Tag library replaces the bbui and bbdata tag libraries

Other References

For more information about user interface design in Blackboard, and using the Blackboard tag libraries, you can find more information in the following documents.

  • Blackboard Learn Design Framework for Release 9.0

Configuration

To use the bbNG tags in your JSP pages, you must include the following Taglib declaration at the top of every pages

<%@ taglib prefix="bbNG" uri="/bbNG"%>
